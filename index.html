First-time setup
Run this SQL once in your Supabase project → SQL Editor:
create table clips ( id uuid default gen_random_uuid() primary key, room text not null, text text not null, device text, tags text[] default '{}', pinned boolean default false, type text default 'text', copy_count integer default 0, created_at timestamptz default now() ); alter table clips enable row level security; create policy "open" on clips for all using (true) with check (true);
